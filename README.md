# Flutter Experiments

Welcome to the **Flutter Experiments** repository!  
This repo contains **10 practical experiments** to help you learn Flutter and Dart step by step. Each experiment focuses on different concepts like widgets, layouts, state management, forms, animations, API integration, responsiveness, and testing.



## Experiments Overview
| [Exp 1](exp1/README.md)   | Install Flutter & Dart SDK, write a simple Dart program |
| [Exp 2](exp2/README.md)   | Explore Flutter widgets, implement Row, Column, Stack layouts |
| [Exp 3](exp3/README.md)   | Design responsive UI, implement media queries & breakpoints |
| [Exp 4](exp4/README.md)   | Setup navigation between screens, use named routes |
| [Exp 5](exp5/README.md)   | Learn Stateful & Stateless widgets, manage state with setState & Provider |
| [Exp 6](exp6/README.md)   | Create custom widgets, apply themes & custom styling |
| [Exp 7](exp7/README.md)   | Design forms with input fields, implement validation |
| [Exp 8](exp8/README.md)   | Add animations to UI elements, experiment with fade & slide effects |
| [Exp 9](exp9/README.md)   | Fetch data from REST API, display data in the UI |
| [Exp 10](exp10/README.md) | Write unit tests, use Flutter debugging tools |
